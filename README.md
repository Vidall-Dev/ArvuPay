## 🚀 Visão Geral e Funcionalidades

O ArvuPay é um sistema desenvolvido para automatizar o gerenciamento de cobrança de mensalidades de associações, reduzindo o trabalho manual e o risco de inadimplência.

### 🎯 Recursos Principais

* **Gestão de Assinaturas:** Cadastro e administração de membros e seus respectivos planos.
* **Controle de Inadimplência:** Dashboards e relatórios para monitorar membros em atraso.
* **Emissão de Faturas:** Processo de geração de faturas simplificado.
* **Rotas Protegidas:** Sistema de autenticação robusto para acesso de administradores.
